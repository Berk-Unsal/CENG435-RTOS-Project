Case Study 6: Shared Network Driver

Extending the ADAS system with Ethernet I/O, a shared driver, and a network stack task. Adjust priority, load, and orchestration to observe successful transmission or contention-driven drops.

Scenario Controls

Network terms (quick definitions)
ISR: interrupt service routine that reacts immediately.
Driver: low-level code that controls the NIC hardware.
Stack: protocol engine (TCP/IP) that formats and routes data.
TX/RX Ring: circular buffers holding outgoing/incoming frames.
MTU: maximum packet size the link can carry without fragmentation.
Backpressure: slowing a producer when queues are full.
Drop: a message discarded when buffers overflow or deadlines expire.
